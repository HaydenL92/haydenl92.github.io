# Hayden Lane Portfolio Website

Personal portfolio website for showcasing software engineering projects, technical skills, and experience.

## Overview

This repository contains the source for my portfolio site. The site is designed to present my work in a more visual, interactive way while giving recruiters and collaborators a quick path to my projects, resume, and technical background.

## Purpose

I built this site to create a stronger online presence beyond LinkedIn and GitHub repository pages. It serves as a central place to highlight:

- software engineering projects
- cloud and data-focused work
- technical skills and stack
- resume and contact information

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## Highlights

- Custom personal portfolio layout
- Mobile-aware presentation
- Project and resume access from a single site
- Static site deployment through GitHub Pages

## Local Preview

You can run the site locally with:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

This site is designed for GitHub Pages deployment.

1. Push the repository to GitHub
2. Open **Settings -> Pages**
3. Set the source to deploy from the `main` branch
4. Publish from the repository root
